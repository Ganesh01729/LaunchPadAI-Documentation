# LaunchPad AI – Smart Business Idea Evaluation System

An AI-powered web application that helps entrepreneurs evaluate and validate startup ideas using **Google Gemini AI**. This repository contains the complete project documentation, including planning, design, development, testing, deployment, and demonstration materials.

---

## About the Project

LaunchPad AI assists entrepreneurs, students, and innovators in assessing their startup ideas by generating:

- Startup Analysis
- SWOT Analysis
- Market Analysis
- Business Recommendations

The project is developed using **Python**, **Flask**, **SQLite**, and **Google Gemini AI**, with deployment on **Render**.
